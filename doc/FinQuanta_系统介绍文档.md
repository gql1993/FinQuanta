# FinQuanta — AI 量化交易平台

## 系统介绍文档

---

## 一、系统概述

**FinQuanta** 是一款基于 PyQt6 构建的桌面端 AI 量化交易平台，集成了多策略选股、AI 智能决策、量子优化、实时行情监控、自动化调度与微信推送等功能，面向 A 股市场，覆盖"数据采集 → 策略研发 → 交易执行 → 风控监控 → 绩效复盘"全流程。

### 核心数据

| 指标 | 数值 |
|------|------|
| 代码规模 | ~16,700 行 Python 代码，50 个模块 |
| 功能模块 | 11 个主功能标签页 |
| 策略体系 | 8 大选股策略（中西方兼顾） |
| 数据库 | 桌面默认 SQLite（`quant.db`，WAL）；服务化 API 可部署为 PostgreSQL（与桌面不要求共库，见 `README.md` / 基础设施说明） |
| AI 仓位 | 5 种并行模拟仓 |
| 定时任务 | 9 项自动化任务 |
| 推送渠道 | Server 酱（微信）+ 企业微信 |

## 三、功能模块详解

### 3.1 📈 总览仪表盘

- **账户摘要卡片**：总资产、浮动盈亏、当日盈亏、持仓数、可用现金、当前回撤
- **市场环境分析**：基于沪深300指数分布日判断市场健康度（🟢健康/🔴偏弱）
- **组合风险速览**：VaR(95%/99%)、最大单股敞口、集中度(HHI)，四色卡片可视化
- **当前持仓表**：实时行情更新，60 秒自动刷新

### 3.2 📡 选股雷达

#### ⭐ 自定义板块
- 板块导航树（行业/概念/自定义三级分类）
- 成分股详情表（价格、涨幅、评分、信号、买入/操作建议）
- **📤 推送强烈买入**：一键推送到微信/企业微信

#### 🔍 扫描选股
**8 大差异化策略**：

| 策略 | 核心逻辑 | 选股偏好 |
|------|----------|----------|
| SEPA (股票魔法师) | 趋势模板 + VCP 收缩 + 突破 | 爆发点 |
| CANSLIM | RS 强度 + 放量新高 + 杯柄形态 | 龙头股 |
| 海龟交易法 | 20/55 日通道突破 + ATR | 趋势突破 |
| 格雷厄姆(价值) | 距高点跌 30%+ + 低于 MA200 | 深度低估 |
| 巴菲特(品质) | 稳健增长 + 低波动 + 均线收敛 | 优质稳定 |
| 彼得林奇(动量) | 月涨 10%+ + 季涨 30%+ | 高动量 |
| 国内趋势 | 多头排列 + 放量突破 + 连阳 | A 股趋势 |
| 国内价值 | 破年线 + 深跌 + 缩量企稳 | A 股抄底 |

#### ⚛️ 量子优化
- **策略融合模式**：8 种量子/经典优化算法（Markowitz+QAOA、量子退火、最小方差、风险平价等）
- **纯量子模式**：QAOA 量子近似优化、模拟退火(量子隧穿)、Tabu 禁忌搜索

#### 🥇 黄金/大宗
- 18 只 ETF 实时行情（黄金、白银、原油、有色金属、煤炭、稀土等）

### 3.3 ⚡ 短期选股

- **事件驱动选股**：输入事件描述（如"伊朗战争"），自动匹配关联板块，推荐受影响个股
- **基金持仓分析**：追踪明星基金经理持仓变化，分析公布前后股价走势

### 3.4 ✅ 走势验证

- 记录每次选股信号，跟踪 3/5/10/20/60 日实际走势
- 自动校准预测准确率
- AI 分析正确/错误原因

### 3.5 💼 手动仓

- 账户总览（资产、盈亏、仓位）
- 持仓表 + **操作栏**（单击显示：明细/买入/卖出/条件单/看行情）
- 买入/卖出表单（自动填充最新价）
- 交易记录

### 3.6 🤖 AI 自主交易模拟仓

**5 种并行模拟仓**：

| 仓位 | 颜色 | 决策方式 |
|------|------|----------|
| 🟣 完全自主仓 | 紫 | AI 全权决策 + 自动执行 |
| 🔴 半自主仓 | 红 | AI 决策，人工触发执行 |
| 🟢 推荐仓 | 绿 | AI 推荐，人工确认执行 |
| 📌 自定义仓 | 橙 | 扫描 Top3 自动跟踪 |
| ⚛️ 量子仓 | 蓝 | QAOA/量子退火优化组合 |

**核心特性**：
- 多 LLM 支持（DeepSeek、GPT、Gemini、Claude、通义千问、Kimi）
- 买入价格强制使用真实市场价（新浪实时 → 日 K 兜底），杜绝 LLM 价格幻觉
- ATR 动态跟踪止损
- 持仓汇总表支持操作栏（明细/买入/卖出/条件单/看行情）
- 双仓对比面板

### 3.7 🦀 OpenClaw 智能体执行网关

**四层架构**：

| 层 | 功能 |
|----|------|
| 📡 感知层 | 多源数据采集、6 项数据源状态监控、实时事件流 |
| 🧠 决策层 | 自然语言策略输入、6 大快捷策略、AI 综合研判 |
| ⚡ 执行层 | 风控参数配置、执行队列、紧急停止 |
| 📊 反馈层 | 6 项绩效卡片、预警日志、AI 策略调优建议 |

**🔄 全流程管线**（一键执行 8 步）：
数据采集 → 因子计算 → AI 研判 → 生成指令 → 风控检查 → 执行交易 → 推送通知 → 记录归因

### 3.8 📊 回测

- 本地 SQLite 数据回测（无需网络）
- 蒙特卡洛模拟
- Walk Forward 滚动验证
- 多策略同时对比

### 3.9 📉 个股分析

- **K 线图**（Plotly.js 渲染）：支持滚轮缩放、拖拽平移、双击重置、日/周/月 K
- Range Slider 时间范围选择器
- 多策略预测叠加（勾选框控制显示）
- 多周期预测校准（5 日/10 日/20 日/1 月/1 季/半年/1 年）
- 数据自动刷新：打开个股时后台自动拉取最新数据

### 3.10 🤖 AI 助手

- 多 LLM 对话（DeepSeek/GPT/Gemini/Claude）
- 读取全部数据库（持仓、策略、事件、基金、回测）进行辅助分析
- 富文本 Markdown 渲染（表格、代码块、彩色列表）
- 历史问答记录

### 3.11 ⚙️ 设置

- **自动化调度引擎**：8 项定时任务，可独立开关
- **消息推送**：Server 酱 + 企业微信双渠道
- **AI 模型配置**：Provider / API Key / Base URL
- **数据管理**：缓存清理、导入/导出
- **外观**：深色/浅色主题

## 四、自动化调度

后台 7×24 运行的 Daemon Scheduler：

| 时间 | 任务 | 说明 |
|------|------|------|
| 09:50 | 拉取实时行情 | 开盘前抓取报价快照 |
| 10:00 | 刷新 K 线日线 | 8 线程并发更新（~800 只/次） |
| 10:05 | 选股雷达扫描 | 多策略扫描，保存 Top50 |
| 10:10 | 短期选股分析 | 事件驱动 + 基金持仓 |
| 10:15 | AI 三仓决策（上午） | 完全自主 + 半自主 + 推荐仓 |
| 13:30 | 刷新 K 线日线 | 下午场前补全 |
| 14:00 | AI 三仓决策（下午） | 第二轮 AI 决策 |
| 15:30 | 走势验证校准 | 对比预测与实际 |
| 15:30 | 日报推送 | 微信/企业微信汇总推送 |
| 全天 5min | 止损止盈预警 | 同一预警每天只推送 1 次 |

## 七、安全与风控

| 措施 | 说明 |
|------|------|
| 价格校验 | 买入价与日 K 收盘价偏差 >15% 自动修正 |
| 资金校验 | 余额不足自动减仓到最大可买手数 |
| T+1 限制 | 当天买入不可当天卖出 |
| ATR 止损 | 动态跟踪止损（2 倍 ATR） |
| 预警去重 | 同一预警每天只推送 1 次 |
| 推送限流 | Server 酱 每天最多 4 次推送 |
| 数据库并发 | WAL 模式 + busy_timeout 防锁死 |

## 八、系统亮点

1. **全策略覆盖**：8 大选股策略真正独立运算，不同策略产出不同候选股
2. **AI + 量子**：LLM 智能决策 + QAOA/量子退火组合优化，双引擎驱动
3. **5 仓并行**：完全自主/半自主/推荐/自定义/量子，多策略实盘对比
4. **全自动化**：数据拉取 → 选股 → 决策 → 执行 → 推送 → 复盘，无人值守
5. **防幻觉**：买入/卖出强制使用真实市场价，杜绝 LLM 价格幻觉
6. **OpenClaw 网关**：四层架构（感知/决策/执行/反馈），一键全流程
7. **交互友好**：单击操作栏 + 双击跳转 + 自动填价 + 滚轮缩放 K 线
