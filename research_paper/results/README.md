# Results

All paper experiment outputs should be written here.

Expected artifacts:

- `metrics.csv`: performance metrics by method.
- `equity_curves.csv`: daily or rebalance-period equity curves.
- `turnover.csv`: turnover and cost breakdown.
- `constraints.jsonl`: optimizer constraints generated per rebalance.
- `llm_cache/`: cached agent responses.
- `figures/`: publication-ready figures.
- `ablation/`: ablation result tables.
- `statistical_tests/`: bootstrap and robustness outputs.

Generated results may become large. Keep only reproducibility-critical small
artifacts in Git unless the project later defines a data/artifact policy.
