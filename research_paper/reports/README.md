# Reports

This directory is reserved for paper-oriented summaries and generated figures.

Planned contents:

- Experiment summary reports.
- Tables for paper drafts.
- Figures for performance, drawdown, turnover, and regime analysis.
- Ablation study summaries.
- Robustness and statistical-test summaries.

Reports should be generated from files in `research_paper/results/`, not from
production trading state.
